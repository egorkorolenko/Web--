Используя структуру сайта, созданную в рамках второй лабораторной работы,
выполнить оформление страницы с сипользованием CSS.

Данная работа будет проверяться только после успешного выполнения второй 
лабораторной работы.

Должны быть выполнены следующие требования:

1. Основная страница сайта (это может быть приветственная страница) должна иметь название index.html
Остальные страницы называйте по своему усмотрению.
2. Сайт должен использовать теги HTML5 для описания структуры.
3. Верстка сайта должна выполняться в "блочном" стиле, без использования таблиц (для игрового поля допускается верстка таблицей).
4. Таблица стилей CSS должна быть подключена как внешний файл.
5. Использование атрибута style внутри html не допускается.
6. Использование атрибутов width, height, align и аналогичных им (то есть имеющих аналоги в CSS) внутри html тегов не допускается.
7. При оформлении текста должны использоваться в том числе и цвета, отличные от черного и белого.
Где именно и какие - на ваше усмотрение.
8. Раздел навигации должен быть заключен в тег <nav> и визуально располагаться слева от основного контента.
9. Заголовок сайта должен располагаться в верхней части страницы, выравнивание по центру, иметь отличный от основного контента фона.
10. При прокрутке основного содержимого страницы заголовок и панель навигации должны оставаться на месте.
11. Предусмотреть на странице игры место для вывода записи партии.
12. Игровое поле должно соответствовать реальному, иметь необходимые подписи строк и столбцов и содержать начальную расстановку фигур.


Исходный код проекта сохранить в репозитории в папке lab3 в ветке master.
Код, расположенный в других ветках или папках, проверяться не будет.
